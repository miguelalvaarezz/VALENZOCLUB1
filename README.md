## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build for Production

```bash
npm run build
npm start
```

## Design System

### Colors

- **Background Dark**: `hsl(230, 20%, 6%)`
- **Blue Primary**: `hsl(217, 90%, 60%)`
- **Blue Glow**: `hsl(217, 100%, 70%)`
- **Accent**: `hsl(240, 100%, 98%)`

### Typography

- **Sans**: Inter (body text)
- **Serif**: Merriweather (headings, quotes)

### Animations

- Fade in
- Slide up
- Glow pulse
- Floating
- Magnetic hover
